# HireHub Static App

Pure HTML/CSS/JS app with 5 screens and local demo persistence.

## Current storage
- Signup/login/profile/role/posts are stored in browser `localStorage`.
- Temporary selected role during signup step is in `sessionStorage`.

## Run
Open `index.html` directly in a browser.

## Implemented features
- Landing, choose role, signup, login, home
- Text-only posts
- Edit posts from profile panel
- Add Education/Experience/Skills as pill UI items
# HireHub Starter

LinkedIn-like starter app with:
- Landing / role selection / signup / login
- Home feed with profile slide-in panel
- Node.js + Express API
- MySQL schema for users and profile data

## Run
1. `cp .env.example .env`
2. Update DB credentials
3. `npm install`
4. `npm run dev`
5. Open `http://localhost:3000`
